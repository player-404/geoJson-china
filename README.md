# getGeoJson

echarts 地图 geoJson 数据下载

### 实现

-   使用高德的行政区等 api 获取地图的 GeoJosn 数据
-   直接在`index.html`中映入高德 AMapUI -使用到的 api
    -   AMapUI 组件中的行政区划浏览

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
