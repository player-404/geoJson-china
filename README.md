# getGeoJson

最新 geoJson 数据下载
[网址](https://player-404.github.io/map-geoJson/)

### 实现

-   使用高德的行政区等 api 获取地图的 GeoJosn 数据
-   直接在`index.html`中引入高德 AMapUI
-   使用到的 api
    -   AMapUI 组件中的行政区划浏览

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
